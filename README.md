### **_Description_**

A PERN stack application for researching and curating DNA sequences and their evolutionary relationships.

### **_Technologies Used_**

PERN Stack

- Postgres
- Express
- React
- Node

BioTech Stack

- BLAST (Nucleotide search)
- MUSCLE (Multiple Sequence Alignment)
- phylotree.js (tree visualization)
